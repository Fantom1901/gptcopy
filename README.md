# 🚀 gptcopy

Консольная утилита для быстрой подготовки контекста проекта для LLM (ChatGPT, Claude, DeepSeek).

Инструмент собирает дерево проекта и содержимое файлов в один блок текста с Markdown-разметкой и автоматически копирует его в буфер обмена.

---

## ✨ Особенности

- **Умное копирование:** Автоматически игнорирует `node_modules`, `.git`, бинарные файлы и lock-файлы.
- **Git-интеграция:** Режим `-c` позволяет копировать только изменённые файлы (идеально для правок в больших проектах).
- **Минификация:** Флаг `-m` удаляет пустые строки, экономя токены и место в окне чата.
- **Авто-детект:** Автоматически определяет `kitten`, `wl-copy` или `xclip` в зависимости от системы и терминала.

---

## 🛠 Установка

Запустите в терминале:

```bash
curl -sSL https://raw.githubusercontent.com/Fantom1901/gptcopy/main/install.sh | bash
```

После установки перезапустите терминал  
или выполните `source` вашего shell-конфига (скрипт подскажет путь).

---

## 🚀 Использование

### Базовые команды

```bash
# Копировать весь текущий проект
gptcopy .

# Копировать конкретную папку с минификацией
gptcopy src/ -m

# Копировать только изменённые в Git файлы
gptcopy -c
```

---

## 🔧 Доступные флаги

| Флаг | Описание |
|------|----------|
| `-c`, `--changed` | Только изменённые / новые файлы из `git status` |
| `-m`, `--minify`  | Удаление пустых строк из кода |
| `-h`, `--help`    | Вызов справки |

---

## 📂 Что игнорируется?

По умолчанию утилита игнорирует:

```
node_modules
.git
venv
dist
build
.next
__pycache__
package-lock.json
```

и прочие служебные или временные файлы.

---

## 💻 Требования

- **OS:** Linux (Arch, Ubuntu и др.) или macOS
- **Shell:** Bash, Zsh или Fish
- **Зависимости:**
    - `curl`
    - `git` (для режима `-c`)
    - `tree` (опционально, для красивой структуры)
